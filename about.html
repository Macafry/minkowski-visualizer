About this Project

Motivation

This website was created as a creative project for my Math 3120 course. I selected a mathematical concept (Minkowski Sum and Difference) and produced something creative around it. In this case, I decided to manifest my creativity through a website with an interactive tool that helps develop some intuiton for Minkowski Sums and Differences.

Design Choices - Convex hull

For this visualizer I decided to let the convex hull of a set of points represent a shape as it's more computationally effcient. An efficient algorithm is very important considering this is an interactive vizualizer. Another reason for which I decided to use the convex hull is that it allows for a bruteforce algorithm. This is particularly nice as, given my time constraints, I couldn't learn about an effcient algorithm to compute these operations.

To calculate the convex hull of a set of points I relied on Andrii Heonia's library.
